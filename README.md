# pos-system
Point of Sale (POS) System using PHP &amp; MySQL


# Features
- User login system
- Product inventory management
- Add products to cart
- Checkout and sales processing

# Technologies Used
- PHP
- MySQL
- HTML
- CSS

# Database
Database name: `pos_project`  
Import `database.sql` into phpMyAdmin before running the project.
